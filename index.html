Etiquetas HTML

Nombre Descripción Ejemplo Info Adicional
HTML El elemento HTML es todo, desde la etiqueta
inicial hasta la etiqueta final		 <html>El contenido va aquí...</html>
Body El elemento define el cuerpo del documento. <body>

<h1>Mi primer título</h1>
<p>Mi primer párrafo</p>

</body>
h1 El elemento define un encabezado. <h1>Título</h1> Se recomienda no tener más de una etiqueta h1,
por asuntos de CEO (búsqueda)
p El elemento define un párrafo. <p>Esto es un párrafo</p>
br La etiqueta define un salto de línea,
y es un elemento vacío sin una etiqueta de cierre		 Esto es un párrafo <br>
con un salto de línea
h2 El elemento define un encabezado de tipo 2. <h2>Título 2</h2>
h3 El elemento define un encabezado de tipo 3. <h3>Título 3</h3>
h4 El elemento define un encabezado de tipo 4. <h4>Título 4</h4>
h5 El elemento define un encabezado de tipo 5. <h5>Título 5</h5>
h6 El elemento define un encabezado de tipo 6. <h6>Título 6</h6>
a La etiqueta define un hipervínculo.
El atributo especifica la dirección URL de la página.		 <a href="https://www.w3schools.com">Visit W3Schools></a>
img La etiqueta se utiliza para incrustar
una imagen en una página HTML.		 <img src="/ruta del recurso" alt="nombre para identificar"> El atributo SRC especifica la ruta de acceso
a la imagen que se va a mostrar
pre El elemento HTML define el texto preformateado. 
<pre>
My Bonnie lies over the ocean.

                
My Bonnie lies over the sea.
              
                
My Bonnie lies over the ocean.
              
                
Oh, bring back my Bonnie to me. 
</pre>
 El texto dentro de un elemento se muestra
en una fuente de ancho fijo (generalmente Courier),
y conserva tanto los espacios
como los saltos de línea
abbr La etiqueta define una abreviatura o un acrónimo. <abbr title="World Health Organization">WHO<abbr>
fue fundada en 1948
address La etiqueta define la información de contacto
del autor/propietario de un documento o un artículo.		 <address>
Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA
</address>
article La etiqueta especifica contenido independiente y autónomo. <article>
<h2>Google Chrome</h2>
<p>Google Chrome is a web browser developed by Google,
released in 2008. Chrome is the world's most
popular web browser today!</p>
</article>		 Un artículo debe tener sentido por sí mismo
y debe ser posible distribuirlo
independientemente del resto del sitio.
aside La etiqueta define algún contenido aparte
del contenido en el que se coloca.		 <aside>
<h4>Epcot Center</h4>
<p>Epcot is a theme park at Walt Disney World Resort
featuring exciting attractions, international pavilions,
award-winning fireworks and seasonal special events.</p>
</aside>
audio La etiqueta se utiliza para incrustar
sonido contenido de un documento,
como música u otras secuencias de audio.		 <audio controls>
<source src="horse.ogg" type="audio/ogg">
<source src="horse.mp3" type="audio/mpeg">
Tu navegador no soporta la etiqueta audio.
<audio>		 El texto entre las etiquetas y
solo se mostrará en los navegadores
que no admitan el elemento.
b La etiqueta especifica el texto en negrita
sin ninguna importancia adicional.		 Este texto es <b>negrita</b> De acuerdo con la especificación HTML5,
la etiqueta debe usarse como ÚLTIMO recurso
cuando ninguna otra etiqueta sea más apropiada.
base La etiqueta especifica la URL base y/o
el destino de todas las URL relativasbr de un documento.		 <base href="https://www.ejemplo.com/" target="_blank">
area Define un área dentro de un mapa de imagen. <area shape="circle" coords="75,75,50" href="pagina.html" alt="Ejemplo"> No tiene etiqueta de cierre.
bdi Aísla parte de texto que pueda tener dirección diferente. <bdi>Texto</bdi>
bdo Controla la dirección del texto. <bdo dir="rtl">Texto</bdo> Para idiomas de derecha a izquierda.
blockquote Representa una cita en bloque. <blockquote>Cita textual</blockquote> Puede incluir el atributo cite.
button Define un botón interactivo. <button type="submit">Enviar</button>
canvas Área para dibujar gráficos mediante scripting (JavaScript). <canvas id="miCanvas" width="200" height="100"></canvas> Necesita JavaScript para mostrar contenido.
caption Define el título o pie de una tabla. <caption>Título de la tabla</caption> Debe ser el primer elemento dentro de table.
cite Indica el título de una obra. <cite>Título</cite>
code Muestra un fragmento de código. <code>print("Hola Mundo")</code>
col Define propiedades para una o más columnas en una tabla. <col span="1" style="width: 100px"> No tiene etiqueta de cierre.
colgroup Agrupa columnas dentro de una tabla. <colgroup><col span="2" style="background-color: yellow"></colgroup>
data Asocia un valor a un contenido para uso de máquinas. <data value="100">Cien</data>
datalist Define una lista de opciones predefinidas para un input. <input list="opciones"><datalist id="opciones"><option value="Opción 1"></datalist>
dd Define la descripción de un término en una lista de definiciones. <dd>Definición</dd>
dfn Indica el término que se define. <dfn>Término</dfn>
del Indica que el texto fue eliminado del contenido. <del>Texto eliminado</del> Puedes usar el atributo cite para referenciar la fuente.
details Muestra contenido adicional que puede ocultarse o mostrarse. <details><summary>Más info</summary>Contenido extra</details>
dialog Define un cuadro de diálogo o ventana modal. <dialog open>Contenido</dialog>
div Elemento de bloque genérico para agrupar contenido. <div>Contenido agrupado</div>
dl Representa una lista de definiciones. <dl><dt>Término</dt><dd>Definición</dd></dl>
dt Define el término a definir en una lista de definiciones. <dt>Término</dt>
em Indica énfasis en el texto. <em>Texto enfatizado</em>
embed Incrusta contenido externo (por ejemplo, multimedia o plugins). <embed src="archivo.swf"> No tiene etiqueta de cierre.
fieldset Agrupa elementos relacionados dentro de un formulario. <fieldset> ... </fieldset>
figcaption Define la leyenda o pie de foto de un elemento figure. <figcaption>Pie de foto</figcaption>
figure Contenedor para contenido ilustrativo (imágenes, diagramas, etc.). <figure><img src="imagen.jpg" alt="Descripción"><figcaption>Pie de foto</figcaption></figure> Se usa junto con figcaption.
footer Define el pie de página o de sección. <footer> ... </footer> Generalmente contiene información de contacto o derechos de autor.
form Define un formulario para recolectar datos del usuario. <form action="/enviar" method="post"> ... </form>
head Contiene metadatos e información sobre el documento. <head> ... </head> Incluye elementos como title, meta, link, etc.
header Define la cabecera de un documento o sección. <header> ... </header> Puede contener logotipos y menús.
hr Representa una separación temática horizontal. <hr> No tiene etiqueta de cierre.
i Texto en cursiva o para términos en otros idiomas. <i>Texto en cursiva</i>
iframe Incrusta otro documento dentro del actual. <iframe src="pagina.html" title="Descripción"></iframe>
input Define un campo de entrada interactivo. <input type="text" name="usuario" placeholder="Usuario"> No tiene etiqueta de cierre.
ins Indica que el texto fue insertado en el contenido. <ins>Texto insertado</ins>
kbd Indica entrada del teclado. <kbd>Ctrl+C</kbd>
label Define una etiqueta para un elemento de formulario. <label for="nombre">Nombre:</label>
legend Define un título para el contenido de un fieldset. <legend>Información personal</legend>
li Representa un elemento de lista. <li>Elemento de lista</li> Debe estar dentro de ol, ul o menu.
link Enlaza recursos externos, como hojas de estilo. <link rel="stylesheet" href="styles.css"> No tiene etiqueta de cierre.
main Representa el contenido principal del documento. <main> ... </main> Debe haber solo uno por página.
map Define un mapa de imagen con áreas clicables. <map name="mapa"><area shape="rect" coords="34,44,270,350" href="pagina.html"></map>
mark Resalta o marca un texto. <mark>Texto resaltado</mark>
menu Define una lista de comandos o un menú de contexto. <menu><li>Opción</li></menu> En HTML5 se usa para menús contextuales.
menuitem Define un ítem en un menú de comandos. <menuitem label="Opción"> Actualmente en desuso en muchos navegadores.
meta Define metadatos sobre el documento, como el juego de caracteres. <meta charset="UTF-8"> No tiene etiqueta de cierre.
meter Representa una medida escalar en un rango conocido. <meter value="0.6"></meter>
nav Define un bloque de enlaces de navegación. <nav><a href="#inicio">Inicio</a></nav> Utilizado para menús de navegación.
noscript Muestra contenido alternativo si JavaScript está deshabilitado. <noscript>Por favor, habilita JavaScript.</noscript> Utilizado para mejorar la accesibilidad.
object Incrusta recursos multimedia u otros documentos. <object data="archivo.pdf" type="application/pdf"></object> Puedes incluir contenido alternativo.
ol Representa una lista ordenada. <ol><li>Elemento</li></ol> Los elementos se numeran.
optgroup Agrupa opciones dentro de un select. <optgroup label="Grupo"><option value="1">Opción 1</option></optgroup>
option Define una opción en un menú select o datalist. <option value="1">Opción 1</option>
output Muestra el resultado de un cálculo o acción. <output name="resultado"></output>
p Representa un párrafo de texto. <p>Este es un párrafo.</p>
picture Contenedor para imágenes con múltiples fuentes. <picture><source srcset="imagen.webp" type="image/webp"><img src="imagen.jpg" alt="Ejemplo"></picture>
progress Representa el progreso de una tarea. <progress value="70" max="100">70%</progress>
q Define una cita corta en línea. <q>Cita en línea</q> Añade comillas automáticamente.
ruby Contenedor para anotaciones ruby (por ejemplo, pronunciación). <ruby>漢<rt>kan</rt></ruby>
rp Proporciona paréntesis de soporte para navegadores sin soporte ruby. <rp>(</rp><rt>kan</rt><rp>)</rp> Opcional.
rt Define la pronunciación en una anotación ruby. <rt>kan</rt> Se usa dentro de ruby.
s Indica que el texto está tachado o ya no es correcto. <s>Texto tachado</s>
samp Muestra la salida de un programa o sistema. <samp>Salida de error</samp>
script Contiene código JavaScript. <script>console.log('Hola');</script> Puede tener atributo src para cargar archivos externos.
section Define una sección genérica en un documento. <section> ... </section> Debe tener un encabezado.
select Define un menú desplegable. <select name="opciones"><option value="1">Opción 1</option></select>
small Reduce el tamaño del texto. <small>Texto pequeño</small>
span Elemento en línea genérico para agrupar o aplicar estilos. <span class="resaltado">Texto</span>
strong Indica una importancia fuerte en el texto. <strong>Texto importante</strong>
style Contiene estilos CSS internos para el documento. <style>body { font-family: Arial; }</style> Se coloca normalmente en head.
sub Texto en subíndice. <sub>2</sub>
sup Texto en superíndice. <sup>2</sup>
table Define una tabla de datos. <table> ... </table>
tbody Define el cuerpo de una tabla. <tbody> ... </tbody>
td Define una celda de datos en una fila de tabla. <td>Dato</td>
template Define contenido inactivo que puede clonarse mediante script. <template><div>Plantilla</div></template> No se renderiza hasta ser utilizada por JavaScript.
textarea Define un área de texto multilinea. <textarea name="mensaje" rows="4" cols="50"></textarea>
tfoot Define el pie de una tabla. <tfoot> ... </tfoot>
th Define una celda de encabezado en una tabla. <th>Encabezado</th>
thead Define el encabezado de una tabla. <thead> ... </thead>
time Representa una fecha u hora. <time datetime="2025-02-07">7 de febrero de 2025</time> Utiliza el atributo datetime.
title Define el título del documento, que se muestra en la pestaña. <title>Mi sitio web</title> Debe estar dentro de head.
track Define pistas de texto para video o audio (subtítulos, descripciones). <track kind="subtitles" srclang="es" label="Español"> No tiene etiqueta de cierre.
u Texto subrayado. <u>Texto subrayado</u>
ul Representa una lista desordenada. <ul><li>Elemento</li></ul> Los elementos utilizan viñetas.
var Muestra una variable en un fragmento de código. <var>x</var>
video Define contenido de video. <video controls><source src="video.mp4" type="video/mp4"></video> Puede incluir múltiples fuentes y subtítulos.
wbr Sugiere un punto de quiebre de línea en palabras largas. <wbr> No tiene etiqueta de cierre.
